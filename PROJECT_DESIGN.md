# Project Design

## Personality
Confident product storyteller: clear, concise, and upbeat. Feels like a PM and DevRel duo turning engineering changes into narratives.

## Colors
- Primary: Electric blue (#2563EB)
- Secondary: Amber accent (#F59E0B)
- Neutrals: Slate (#0F172A text, #1F2937 panels, #E5E7EB borders)
- Success: Emerald (#10B981)

## Typography
- Headings: Inter, semibold
- Body: Inter, regular
- Monospace accents for snippets: JetBrains Mono

## Components
- Input panel: textarea/dropzone, URL helper, "use sample release" button
- Change list: grouped cards with tags (customer/internal/recruiter), checkboxes for inclusion
- Demo flow: numbered steps with timing and speaking notes
- Launch recap: markdown-ish block with sections for customer changelog and internal notes
- Social thread: stacked tweet cards with character count
- Actions: copy buttons, "regenerate" per section, export to Markdown

## Layout
Two-column responsive layout: left for inputs and grouping controls, right for outputs with tabs (Demo, Changelog, Launch thread). Persistent top bar for audience switcher and sample-data toggle.

## Inspiration
- Linear and Vercel changelog cadence for clarity
- Notion-style clean blocks for editability
- Slide decks with step-by-step demo cues
