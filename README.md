# changelog-to-demo

Bootstrap for a Next.js + TypeScript + Tailwind app that turns release notes into demo-ready narratives.

## Scripts

- `npm run dev` — start local dev server
- `npm run lint` — run Next.js ESLint checks
- `npm run build` — production build check

## Current scope (slice-000)

- App Router project scaffold with strict TypeScript
- Tailwind styling wired through PostCSS
- Intentional top bar + two-column responsive layout based on `PROJECT_DESIGN.md`
- Starter UI for input grouping and output tabs (Demo / Changelog / Launch thread)
